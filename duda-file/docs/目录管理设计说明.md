# 目录管理设计说明

## 概述

基于阿里云OSS的目录管理文档,本文档整理了OSS目录的概念、创建、重命名、删除、查询统计、权限管理及性能优化等内容,为后续统一设计提供基础。

## 一、OSS目录的本质

### 1.1 核心概念

**重要概念**: OSS本质上是扁平存储,**没有真实的文件夹**。

**目录的实现方式**:
- OSS通过文件名中的"/"分隔符模拟文件夹
- 控制台显示的"目录"是根据Object的Key自动生成的
- 目录实际上是一个以"/"结尾的0字节文件(空目录占位符)

**示例**:

**显示的目录结构**:
```
examplebucket
    └── log/
       ├── date1.txt
       ├── date2.txt
       ├── date3.txt
    └── destfolder/
       └── 2021/
          └── photo.jpg
```

**实际存储的Object**:
```
log/date1.txt
log/date2.txt
log/date3.txt
destfolder/2021/photo.jpg
```

### 1.2 目录的两种生成方式

**方式1: 上传文件时自动生成**
```java
// 上传包含路径的文件时,目录自动出现
String objectKey = "log/date1.txt";
ossClient.putObject(bucketName, objectKey, content);

// 结果: log/目录自动生成
```

**方式2: 主动创建空目录**
```java
// 创建一个以"/"结尾的0字节文件作为目录占位符
String dirName = "exampledir/";
String content = ""; // 空文件

ossClient.putObject(bucketName, dirName,
    new ByteArrayInputStream(content.getBytes()));
```

### 1.3 目录 vs 文件的判断

**判断逻辑**:

## 二、创建目录

### 2.1 创建空目录

**Java实现**:
```java
public static void createDirectory() {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
    String bucketName = "examplebucket";
    String directoryName = "exampledir/"; // 必须以"/"结尾

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    try {
        // 创建空内容
        String content = "";

        // 创建PutObjectRequest
        PutObjectRequest request = new PutObjectRequest(
            bucketName,
            directoryName,
            new ByteArrayInputStream(content.getBytes())
        );

        // 可选: 设置元数据
        ObjectMetadata metadata = new ObjectMetadata();
        metadata.setContentType("application/octet-stream");
        metadata.setObjectAcl(CannedAccessControlList.Private);
        request.setMetadata(metadata);

        // 上传空文件(创建目录)
        ossClient.putObject(request);

        System.out.println("目录创建成功: " + directoryName);

    } catch (Exception e) {
        System.err.println("目录创建失败: " + e.getMessage());
    } finally {
        ossClient.shutdown();
    }
}
```

```java
public static void createDirectoryRecursive(String path) {
    if (!path.endsWith("/")) {
        path += "/";
    }

    String[] parts = path.split("/");
    StringBuilder currentPath = new StringBuilder();

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    try {
        for (int i = 0; i < parts.length; i++) {
            if (parts[i].isEmpty()) continue;

            currentPath.append(parts[i]).append("/");

            try {
                // 尝试创建目录(如果已存在会忽略)
                String content = "";
                PutObjectRequest request = new PutObjectRequest(
                    bucketName,
                    currentPath.toString(),
                    new ByteArrayInputStream(content.getBytes())
                );

                ossClient.putObject(request);
                System.out.println("创建目录: " + currentPath);

            } catch (OSSException oe) {
                // 忽略已存在的错误
                if (!oe.getErrorCode().equals("BucketAlreadyExists")) {
                    // 其他错误抛出
                    throw oe;
                }
            }
        }

    } finally {
        ossClient.shutdown();
    }
}
```

```java
public static boolean directoryExists(String bucketName, String directoryName) {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    try {
        // 方式1: 检查目录文件是否存在

        }

        ossClient.shutdown();
    }
}
```

```java
public static void listDirectoryContents() {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
    String bucketName = "examplebucket";
    String directoryPrefix = "log/"; // 目录前缀

        for (OSSObjectSummary objectSummary : objectSummaries) {
            System.out.println("  " + objectSummary.getKey() +
                " (" + objectSummary.getSize() + " bytes)");
        }

    } finally {
        ossClient.shutdown();
    }
}
```

### 3.2 递归列举目录下所有文件

```java
public static List<OSSObjectSummary> listAllFilesInDirectory(
    String bucketName,
    String directoryPrefix
) {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    List<OSSObjectSummary> allFiles = new ArrayList<>();

    try {
        if (!directoryPrefix.endsWith("/")) {
            directoryPrefix += "/";
        }

        String nextMarker = null;

        do {
            ListObjectsRequest listRequest = new ListObjectsRequest(bucketName);
            listRequest.setPrefix(directoryPrefix);
            listRequest.setMaxKeys(1000);

            // 不设置Delimiter,实现递归列举
            if (nextMarker != null) {
                listRequest.setMarker(nextMarker);
            }

            ListObjectsResult result = ossClient.listObjects(listRequest);

            allFiles.addAll(result.getObjectSummaries());
            nextMarker = result.getNextMarker();

        } while (result.isTruncated());

    } finally {
        ossClient.shutdown();
    }

    return allFiles;
}
```

## 四、重命名目录

### 4.1 重命名目录的原理

**重要**: OSS不支持直接重命名目录

**重命名目录的复合工作流**:
```
1. 复制原目录下的所有文件到新目录
   ↓
2. 确认复制成功后,删除原目录及所有文件
```

**注意**:
- 对大目录非常耗时
- 会产生大量API调用和流量费用
- 请务必谨慎评估

### 4.2 重命名目录实现

```java
public static void renameDirectory(
    String bucketName,
    String oldDirectoryName,
    String newDirectoryName
) {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    try {
        // 确保目录名以"/"结尾
        if (!oldDirectoryName.endsWith("/")) {
            oldDirectoryName += "/";
        }
        if (!newDirectoryName.endsWith("/")) {
            newDirectoryName += "/";
        }

        System.out.println("开始重命名目录: " + oldDirectoryName + " -> " + newDirectoryName);

        // 1. 递归列举原目录下的所有文件
        List<OSSObjectSummary> allFiles = listAllFilesInDirectory(
            bucketName,
            oldDirectoryName
        );

        System.out.println("找到 " + allFiles.size() + " 个文件");

        // 2. 复制所有文件到新目录
        int successCount = 0;
        int failCount = 0;

        for (OSSObjectSummary file : allFiles) {
            String oldKey = file.getKey();
            String newKey = oldKey.replace(oldDirectoryName, newDirectoryName);

            try {
                // 拷贝文件
                CopyObjectRequest copyRequest = new CopyObjectRequest(
                    bucketName,
                    oldKey,
                    bucketName,
                    newKey
                );

                CopyObjectResult copyResult = ossClient.copyObject(copyRequest);
                successCount++;

                System.out.println("复制成功: " + oldKey + " -> " + newKey);

            } catch (Exception e) {
                failCount++;
                System.err.println("复制失败: " + oldKey + ", 错误: " + e.getMessage());
            }
        }

        System.out.println("复制完成, 成功: " + successCount + ", 失败: " + failCount);

        // 3. 删除原目录的所有文件
        if (failCount == 0) {
            System.out.println("开始删除原目录...");

            for (OSSObjectSummary file : allFiles) {
                try {
                    ossClient.deleteObject(bucketName, file.getKey());
                    System.out.println("已删除: " + file.getKey());
                } catch (Exception e) {
                    System.err.println("删除失败: " + file.getKey());
                }
            }

            // 删除目录占位符文件
            try {
                ossClient.deleteObject(bucketName, oldDirectoryName);
                System.out.println("目录占位符已删除");
            } catch (Exception e) {
                // 忽略,目录可能没有占位符
            }

            System.out.println("目录重命名完成!");
        } else {
            System.err.println("部分文件复制失败,取消删除操作");
        }

    } finally {
        ossClient.shutdown();
    }
}
```

### 4.3 优化的目录重命名方案

**改进点**:
1. 使用并发复制提高速度
2. 记录进度,支持断点续传
3. 失败重试机制
4. 提供回滚功能

```java
@Service
public class DirectoryRenameService {

    @Autowired
    private OSSClient ossClient;

    @Autowired
    private RenameTaskRepository renameTaskRepository;

    /**
     * 异步重命名目录
     */
    @Async
    public void renameDirectoryAsync(
        String bucketName,
        String oldDirectoryName,
        String newDirectoryName,
        Long taskId
    ) {
        RenameTask task = renameTaskRepository.findById(taskId).orElseThrow();

        try {
            // 1. 递归列举所有文件
            List<OSSObjectSummary> allFiles = listAllFilesInDirectory(
                bucketName,
                oldDirectoryName
            );

            task.setTotalFiles(allFiles.size());
            task.setStatus("COPYING");
            renameTaskRepository.save(task);

            // 2. 并发复制文件
            int threadPoolSize = 10;
            ExecutorService executor = Executors.newFixedThreadPool(threadPoolSize);

            List<Future<RenameResult>> futures = new ArrayList<>();

            for (OSSObjectSummary file : allFiles) {
                Future<RenameResult> future = executor.submit(() -> {
                    String oldKey = file.getKey();
                    String newKey = oldKey.replace(oldDirectoryName, newDirectoryName);

                    try {
                        // 拷贝文件
                        CopyObjectRequest copyRequest = new CopyObjectRequest(
                            bucketName,
                            oldKey,
                            bucketName,
                            newKey
                        );

                        CopyObjectResult result = ossClient.copyObject(copyRequest);

                        return new RenameResult(oldKey, newKey, true, null);

                    } catch (Exception e) {
                        return new RenameResult(oldKey, newKey, false, e.getMessage());
                    }
                });

                futures.add(future);
            }

            // 等待所有复制任务完成
            int successCount = 0;
            int failCount = 0;
            List<String> failedFiles = new ArrayList<>();

            for (Future<RenameResult> future : futures) {
                try {
                    RenameResult result = future.get();
                    if (result.isSuccess()) {
                        successCount++;
                    } else {
                        failCount++;
                        failedFiles.add(result.getOldKey());
                    }
                } catch (Exception e) {
                    failCount++;
                }
            }

            executor.shutdown();

            // 3. 更新任务状态
            task.setCopiedFiles(successCount);
            task.setFailedFiles(failCount);
            renameTaskRepository.save(task);

            // 4. 如果全部成功,删除原文件
            if (failCount == 0) {
                task.setStatus("DELETING");
                renameTaskRepository.save(task);

                // 批量删除原文件
                deleteObjectsInBatches(bucketName, allFiles);

                // 删除目录占位符
                try {
                    ossClient.deleteObject(bucketName, oldDirectoryName);
                } catch (Exception ignored) {
                }

                task.setStatus("COMPLETED");
                task.setCompletedAt(LocalDateTime.now());

            } else {
                task.setStatus("PARTIAL_FAILED");
                task.setError("部分文件复制失败: " + failedFiles);
            }

            renameTaskRepository.save(task);

        } catch (Exception e) {
            task.setStatus("FAILED");
            task.setError(e.getMessage());
            renameTaskRepository.save(task);
        }
    }

    private void deleteObjectsInBatches(
        String bucketName,
        List<OSSObjectSummary> files
    ) {
        int batchSize = 1000;
        int totalBatches = (files.size() + batchSize - 1) / batchSize;

        for (int i = 0; i < totalBatches; i++) {
            int fromIndex = i * batchSize;
            int toIndex = Math.min(fromIndex + batchSize, files.size());

            List<String> batchKeys = files.subList(fromIndex, toIndex)
                .stream()
                .map(OSSObjectSummary::getKey)
                .collect(Collectors.toList());

            // 批量删除
            DeleteObjectsRequest deleteRequest = new DeleteObjectsRequest(bucketName);
            deleteRequest.setKeys(batchKeys);

            ossClient.deleteObjects(deleteRequest);
        }
    }

    private static class RenameResult {
        private String oldKey;
        private String newKey;
        private boolean success;
        private String error;

        // constructors, getters
    }
}
```

### 5.2 删除非空目录

**重要**: 删除目录会同步删除目录下的所有子目录和文件,请谨慎操作!

```java
public static void deleteDirectoryWithContents(
    String bucketName,
    String directoryPrefix
) {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    try {
        // 确保目录名以"/"结尾
        if (!directoryPrefix.endsWith("/")) {
            directoryPrefix += "/";
        }

        System.out.println("开始删除目录: " + directoryPrefix);

        // 1. 递归列举目录下的所有文件
        List<String> allKeys = new ArrayList<>();
        String nextMarker = null;

        do {
            ListObjectsRequest listRequest = new ListObjectsRequest(bucketName);
            listRequest.setPrefix(directoryPrefix);
            listRequest.setMaxKeys(1000);

            if (nextMarker != null) {
                listRequest.setMarker(nextMarker);
            }

            ListObjectsResult result = ossClient.listObjects(listRequest);

            // 添加所有Object Key
            allKeys.addAll(result.getObjectSummaries().stream()
                .map(OSSObjectSummary::getKey)
                .collect(Collectors.toList()));

            nextMarker = result.getNextMarker();

        } while (result.isTruncated());

        System.out.println("找到 " + allKeys.size() + " 个对象需要删除");

        // 2. 批量删除
        int batchSize = 1000;
        int totalBatches = (allKeys.size() + batchSize - 1) / batchSize;

        int deletedCount = 0;

        for (int i = 0; i < totalBatches; i++) {
            int fromIndex = i * batchSize;
            int toIndex = Math.min(fromIndex + batchSize, allKeys.size());

            List<String> batchKeys = allKeys.subList(fromIndex, toIndex);

            DeleteObjectsRequest deleteRequest = new DeleteObjectsRequest(bucketName);
            deleteRequest.setKeys(batchKeys);

            DeleteObjectsResult deleteResult = ossClient.deleteObjects(deleteRequest);

            deletedCount += deleteResult.getDeletedObjects().size();

            System.out.printf("批次 %d/%d 删除完成%n", i + 1, totalBatches);
        }

        // 3. 删除目录占位符
        try {
            ossClient.deleteObject(bucketName, directoryPrefix);
            System.out.println("目录占位符已删除");
        } catch (Exception e) {
            // 目录占位符可能不存在
            System.out.println("目录占位符不存在或已删除");
        }

        System.out.println("目录删除完成, 共删除 " + deletedCount + " 个对象");

    } finally {
        ossClient.shutdown();
    }
}
```

### 5.4 使用生命周期规则自动删除

**场景**: 定期清理过期目录

```java
public static void setLifecycleRuleForDirectory() {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
    String bucketName = "examplebucket";

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    try {
        // 创建生命周期规则
        LifecycleRule rule = new LifecycleRule("delete-temp-directory");

        // 删除temp/目录下30天前的文件
        rule.setPrefix("temp/");
        rule.setExpirationDays(30);

        // 也可以基于目录标签删除
```

### 8.5 预热热门目录

**场景**: 某些目录访问非常频繁

**方案**: 将热门目录的文件列表缓存到内存

```java
@Service
public class HotDirectoryCache {

    private final LoadingCache<String, List<String>> cache;

    public HotDirectoryCache() {
        this.cache = CacheBuilder.newBuilder()
            .maximumSize(1000) // 最多缓存1000个目录
            .expireAfterWrite(10, TimeUnit.MINUTES) // 10分钟过期
            .build(new CacheLoader<String, List<String>>() {
                @Override
                public List<String> load(String key) throws Exception {
                    // 从OSS加载目录列表
                    String[] parts = key.split(":");
                    String bucketName = parts[0];
                    String directory = parts[1];

                    return listFilesFromOSS(bucketName, directory);
                }
            });
    }

    public List<String> getHotDirectory(String bucketName, String directory) {
        String key = bucketName + ":" + directory;
        return cache.getUnchecked(key);
    }

    public void invalidate(String bucketName, String directory) {
        cache.invalidate(bucketName + ":" + directory);
    }
}
```

### 10.4 如何遍历所有目录?

```java
public static List<String> listAllDirectories(String bucketName) {
    String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";

    OSS ossClient = new OSSClientBuilder().build(endpoint);

    List<String> directories = new ArrayList<>();

    try {
        String nextMarker = null;

        do {
            ListObjectsRequest listRequest = new ListObjectsRequest(bucketName);
            listRequest.setDelimiter("/");
            listRequest.setMaxKeys(1000);

            if (nextMarker != null) {
                listRequest.setMarker(nextMarker);
            }

            ListObjectsResult result = ossClient.listObjects(listRequest);

            // 添加所有CommonPrefixes(都是目录)
            directories.addAll(result.getCommonPrefixes());

            nextMarker = result.getNextMarker();

        } while (result.isTruncated());

    } finally {
        ossClient.shutdown();
    }

    return directories;
}
```
