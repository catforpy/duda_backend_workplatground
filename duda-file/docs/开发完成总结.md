## 一、项目概述

**duda-file** 是一个统一的文件管理服务模块,支持多云存储(阿里云OSS、腾讯云COS、七牛云Kodo、MinIO等),提供完整的文件管理功能。

**核心功能**:
- Bucket管理 (创建、删除、配置、统计)
- 文件上传下载 (简单上传、分片上传、断点续传)
- 客户端直传 (STS、表单上传、预签名URL)
- 文件管理 (删除、复制、重命名、恢复)
- 目录操作 (创建、删除、重命名、统计)
- 权限控制 (三级权限、配额管理)

### 4.2 设计模式应用

| 设计模式 | 应用场景 | 位置 |
|----------|----------|------|
| **适配器模式** | 多云适配器 | StorageService接口及实现 |
| **工厂模式** | 适配器创建 | StorageAdapterFactory |
| **策略模式** | 上传/下载策略 | UploadStrategy, DownloadStrategy |
| **门面模式** | 服务封装 | BucketService, ObjectService |
| **模板方法** | 业务流程 | Manager层统一处理流程 |

### 4.3 核心特性

✅ **多云支持**: 统一接口支持阿里云OSS、腾讯云COS、七牛云、MinIO
✅ **适配器模式**: 屏蔽云厂商差异,易于扩展
✅ **权限控制**: 三级权限(Bucket级、Object级、系统级)
✅ **配额管理**: 用户级、Bucket级、单文件级配额
✅ **目录操作**: 通过前缀模拟完整的目录操作
✅ **断点续传**: 支持大文件分片上传和断点续传
✅ **客户端直传**: STS、表单上传、预签名URL
✅ **RPC服务**: 标准的Dubbo服务,支持分布式部署

## 七、使用指南

### 7.1 快速开始

**1. 引入依赖**:
```xml
<dependency>
    <groupId>com.duda</groupId>
    <artifactId>duda-file-api</artifactId>
    <version>1.0.0</version>
</dependency>
<dependency>
    <groupId>com.duda</groupId>
    <artifactId>duda-file-provider</artifactId>
    <version>1.0.0</version>
</dependency>
```

**2. 配置Dubbo服务**:
```xml
<dubbo:reference id="bucketService"
    interface="com.duda.file.service.BucketService" />
<dubbo:reference id="objectService"
    interface="com.duda.file.service.ObjectService" />
<dubbo:reference id="uploadService"
    interface="com.duda.file.service.UploadService" />
<dubbo:reference id="downloadService"
    interface="com.duda.file.service.DownloadService" />
```

**3. 调用服务**:
```java
// 创建Bucket
BucketDTO bucket = bucketService.createBucket(request);

// 上传文件
UploadResultDTO result = uploadService.simpleUpload(request);

// 下载文件
DownloadResultDTO result = downloadService.download(request);
```
