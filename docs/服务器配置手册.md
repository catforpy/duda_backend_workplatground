📝 配置步骤

1. 登录 https://ecs.console.aliyun.com/
2. 找到你的 ECS 实例（IP: 120.26.170.213）
3. 点击 "安全组" → "配置规则"
4. 点击 "手动添加"
5. 填写规则：
   规则方向：入方向
   授权策略：允许
   协议类型：自定义TCP
   端口范围：8848/8848
   授权对象：0.0.0.0/0
   优先级：1
   描述：Duda Nacos
6. 重复步骤4-5，添加所有需要开放的端口
