## 六、Spring Boot 数据源配置示例

### 6.1 单数据源配置（bootstrap.yml）

```yaml
spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://120.26.170.213:3306/duda_nexus?useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: duda2024
    type: com.alibaba.druid.pool.DruidDataSource
```

### 6.2 主从数据源配置（如需读写分离）

```yaml
spring:
  datasource:
    dynamic:
      primary: master  # 默认数据源
      strict: false
      datasource:
        master:
          url: jdbc:mysql://120.26.170.213:3306/duda_nexus?useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
          username: root
          password: duda2024
          driver-class-name: com.mysql.cj.jdbc.Driver
        slave:
          url: jdbc:mysql://120.26.170.213:3307/duda_nexus?useSSL=false&serverTimezone=Asia/Shanghai
          username: root
          password: ""
          driver-class-name: com.mysql.cj.jdbc.Driver
```

## 九、安全建议

1. ⚠️ **生产环境必须修改默认密码**
2. 建议配置防火墙白名单，限制访问来源
3. 敏感配置使用环境变量或密钥管理工具
4. 从库建议设置密码，不要使用空密码
5. 定期备份数据库

## 十、故障排查

### 常见问题

1. **连接超时**
   - 检查防火墙规则
   - 确认云服务器安全组配置

2. **认证失败**
   - 验证用户名密码
   - 检查 MySQL 用户权限

3. **主从同步延迟**
   - 检查从库同步状态
   - 查看主库 Binlog 日志
