# 公共模块使用指南

## 📦 已创建的公共模块

### 1. 工具类（Util）

#### IdGenerator - ID生成器

**位置：** `com.duda.common.util.IdGenerator`

**功能：** 生成全局唯一的分布式ID（雪花算法）

**使用方法：**
```java
// 生成ID
Long id = IdGenerator.nextId();

// 示例：保存用户时生成ID
UserPO userPO = new UserPO();
userPO.setId(IdGenerator.nextId());
userMapper.insert(userPO);
```

---

#### BeanUtils - 对象转换工具

**位置：** `com.duda.common.util.BeanUtils`

**功能：** 对象属性拷贝（PO、DTO、VO之间转换）

**使用方法：**
```java
// 1. 单个对象转换
UserPO userPO = userMapper.selectById(1L);
UserDTO userDTO = BeanUtils.copy(userPO, UserDTO.class);

// 2. 列表转换
List<UserPO> userPOList = userMapper.selectList(null);
List<UserDTO> userDTOList = BeanUtils.copyList(userPOList, UserDTO.class);

// 3. 分页转换
Page<UserPO> page = userMapper.selectPage(new Page<>(1, 10), null);
PageResult<UserDTO> pageResult = BeanUtils.copyPage(page, UserDTO.class);
```

---

### 2. 领域对象（Domain）

#### Result - 统一响应结果

**位置：** `com.duda.common.domain.Result`

**功能：** 统一API接口的返回格式

**使用方法：**
```java
// 成功返回（无数据）
return Result.success();

// 成功返回（带数据）
return Result.success(userDTO);

// 成功返回（自定义消息）
return Result.success("查询成功", userDTO);

// 失败返回（系统错误）
return Result.error();
return Result.error("系统异常");

// 失败返回（业务错误）
return Result.bizError("用户不存在");

// 失败返回（参数错误）
return Result.errorParam();
return Result.errorParam("用户名不能为空");
```

**返回格式：**
```json
{
  "code": 200,
  "message": "success",
  "data": {...},
  "timestamp": 1735660800000
}
```

---

#### PageResult - 分页结果

**位置：** `com.duda.common.domain.PageResult`

**功能：** 包装分页查询结果

**使用方法：**
```java
// Controller
@GetMapping("/list")
public Result<PageResult<UserDTO>> list(PageQuery query) {
    Page<UserPO> page = userMapper.selectPage(
        new Page<>(query.getCurrent(), query.getSize()),
        null
    );
    PageResult<UserDTO> pageResult = BeanUtils.copyPage(page, UserDTO.class);
    return Result.success(pageResult);
}
```

**返回格式：**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "records": [...],
    "total": 100,
    "current": 1,
    "size": 10,
    "pages": 10
  }
}
```

---

#### PageQuery - 分页查询DTO

**位置：** `com.duda.common.domain.PageQuery`

**功能：** 接收分页查询参数

**使用方法：**
```java
// Controller
@PostMapping("/list")
public Result<PageResult<UserDTO>> list(@RequestBody PageQuery query) {
    // query.getCurrent() - 当前页码（默认1）
    // query.getSize() - 每页条数（默认10，最大100）

    Page<UserPO> page = userMapper.selectPage(
        new Page<>(query.getCurrent(), query.getSize()),
        null
    );
    return Result.success(BeanUtils.copyPage(page, UserDTO.class));
}
```

### 3. 实体类（Entity）

#### BaseEntity - 基础实体

**位置：** `com.duda.common.entity.BaseEntity`

**功能：** 所有实体类的父类，包含公共字段

**包含字段：**
- `id` - 主键ID（雪花算法生成）
- `createTime` - 创建时间（自动填充）
- `updateTime` - 更新时间（自动填充）
- `deleted` - 逻辑删除标志（0=未删除，1=已删除）
- `version` - 版本号（乐观锁）

**使用方法：**
```java
@Data
@TableName("t_user")
public class UserPO extends BaseEntity {
    private String username;
    private String password;
    // 不需要再定义 id、createTime、updateTime、deleted、version
}
```

### 5. 异常处理（Exception）

#### BizException - 业务异常

**位置：** `com.duda.common.web.exception.BizException`

**功能：** 业务逻辑中的异常抛出

**使用方法：**
```java
// 抛出业务异常
if (user == null) {
    throw new BizException("用户不存在");
}

// 自定义错误码
throw new BizException(501, "用户名已存在");

// 在Service中使用
public void deleteUser(Long userId) {
    UserPO user = userMapper.selectById(userId);
    if (user == null) {
        throw new BizException("用户不存在");
    }
    userMapper.deleteById(userId);
}
```

**全局异常处理器会自动捕获并返回：**
```json
{
  "code": 501,
  "message": "用户不存在",
  "data": null
}
```

### 6. 配置类（Config）

#### MyMetaObjectHandler - 字段自动填充

**位置：** `com.duda.common.database.config.MyMetaObjectHandler`

**功能：** 自动填充 `createTime`、`updateTime`、`version`

**使用方法：**
```java
// 在实体类中使用
@TableField(fill = FieldFill.INSERT)
private LocalDateTime createTime;

@TableField(fill = FieldFill.INSERT_UPDATE)
private LocalDateTime updateTime;

// 插入时自动填充createTime、updateTime、version
// 更新时自动填充updateTime
```

---

#### ResponseAdvisor - 全局响应增强

**位置：** `com.duda.common.web.config.ResponseAdvisor`

**功能：** 自动将Controller返回值包装为Result格式

**使用方法：**
```java
// Controller中不需要手动包装
@GetMapping("/user/{id}")
public UserDTO getUser(@PathVariable Long id) {
    UserPO userPO = userMapper.selectById(id);
    // 自动包装为 Result.success(userDTO)
    return BeanUtils.copy(userPO, UserDTO.class);
}
```

**跳过包装：**
```java
// 使用@IgnoreResponseWrapper注解跳过自动包装
@IgnoreResponseWrapper
@GetMapping("/raw-data")
public Object rawData() {
    return "原始数据";
}
```

---

#### GlobalExceptionHandler - 全局异常处理

**位置：** `com.duda.common.web.exception.GlobalExceptionHandler`

**功能：** 统一处理所有异常，返回标准格式

**处理的异常：**
- `BizException` - 业务异常
- `MethodArgumentNotValidException` - 参数校验异常
- `BindException` - 参数绑定异常
- `IllegalArgumentException` - 非法参数异常
- `RuntimeException` - 运行时异常
- `Exception` - 所有未捕获的异常

---

#### CorsConfig - 跨域配置

**位置：** `com.duda.common.web.config.CorsConfig`

**功能：** 允许跨域请求

## 🎯 开发流程示例

### 1. 创建实体类

```java
package com.duda.user.entity;

import com.baomidou.mybatisplus.annotation.TableName;
import com.duda.common.entity.BaseEntity;
import lombok.Data;
import lombok.EqualsAndHashCode;

@Data
@EqualsAndHashCode(callSuper = true)
@TableName("t_user")
public class UserPO extends BaseEntity {
    private String username;
    private String password;
    private String nickname;
    private Integer status;
}
```

### 2. 创建DTO

```java
package com.duda.user.dto;

import lombok.Data;

@Data
public class UserDTO {
    private Long id;
    private String username;
    private String nickname;
    private Integer status;
    private LocalDateTime createTime;
}
```

### 3. 创建Mapper

```java
package com.duda.user.mapper;

import com.baomidou.mybatisplus.core.mapper.BaseMapper;
import com.duda.user.entity.UserPO;
import org.apache.ibatis.annotations.Mapper;

@Mapper
public interface UserMapper extends BaseMapper<UserPO> {
}
```

### 4. 创建Service

```java
package com.duda.user.service;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.baomidou.mybatisplus.core.metadata.IPage;
import com.baomidou.mybatisplus.extension.plugins.pagination.Page;
import com.duda.common.domain.PageQuery;
import com.duda.common.domain.PageResult;
import com.duda.common.util.BeanUtils;
import com.duda.common.util.IdGenerator;
import com.duda.user.dto.UserDTO;
import com.duda.user.entity.UserPO;
import com.duda.user.mapper.UserMapper;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.List;

@Service
public class UserService {

    @Autowired
    private UserMapper userMapper;

    /**
     * 创建用户
     */
    public void createUser(UserDTO userDTO) {
        // 检查用户名是否已存在
        LambdaQueryWrapper<UserPO> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(UserPO::getUsername, userDTO.getUsername());
        if (userMapper.selectCount(wrapper) > 0) {
            throw new BizException("用户名已存在");
        }

        // 转换DTO为PO
        UserPO userPO = BeanUtils.copy(userDTO, UserPO.class);

        // 生成ID
        userPO.setId(IdGenerator.nextId());

        // 保存
        userMapper.insert(userPO);
    }

    /**
     * 根据ID查询用户
     */
    public UserDTO getUserById(Long id) {
        UserPO userPO = userMapper.selectById(id);
        if (userPO == null) {
            throw new BizException("用户不存在");
        }
        return BeanUtils.copy(userPO, UserDTO.class);
    }

    /**
     * 分页查询用户列表
     */
    public PageResult<UserDTO> listUsers(PageQuery query) {
        // 构建分页对象
        Page<UserPO> page = new Page<>(query.getCurrent(), query.getSize());

        // 查询
        IPage<UserPO> userPOPage = userMapper.selectPage(page, null);

        // 转换为DTO
        return BeanUtils.copyPage(userPOPage, UserDTO.class);
    }

    /**
     * 更新用户
     */
    public void updateUser(UserDTO userDTO) {
        UserPO userPO = userMapper.selectById(userDTO.getId());
        if (userPO == null) {
            throw new BizException("用户不存在");
        }

        // 拷贝属性（忽略null值）
        BeanUtils.update(userDTO, userPO);

        // 更新
        userMapper.updateById(userPO);
    }

    /**
     * 删除用户
     */
    public void deleteUser(Long id) {
        UserPO userPO = userMapper.selectById(id);
        if (userPO == null) {
            throw new BizException("用户不存在");
        }

        // 逻辑删除
        userMapper.deleteById(id);
    }
}
```

### 5. 创建Controller

```java
package com.duda.user.controller;

import com.duda.common.domain.PageQuery;
import com.duda.common.domain.PageResult;
import com.duda.common.domain.Result;
import com.duda.user.dto.UserDTO;
import com.duda.user.service.UserService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/user")
public class UserController {

    @Autowired
    private UserService userService;

    /**
     * 创建用户
     */
    @PostMapping("/create")
    public Result<Void> create(@RequestBody UserDTO userDTO) {
        userService.createUser(userDTO);
        return Result.success();
    }

    /**
     * 根据ID查询用户
     */
    @GetMapping("/{id}")
    public UserDTO getUserById(@PathVariable Long id) {
        // 全局响应增强会自动包装为Result
        return userService.getUserById(id);
    }

    /**
     * 分页查询用户列表
     */
    @PostMapping("/list")
    public Result<PageResult<UserDTO>> list(@RequestBody PageQuery query) {
        return Result.success(userService.listUsers(query));
    }

    /**
     * 更新用户
     */
    @PostMapping("/update")
    public Result<Void> update(@RequestBody UserDTO userDTO) {
        userService.updateUser(userDTO);
        return Result.success();
    }

    /**
     * 删除用户
     */
    @DeleteMapping("/{id}")
    public Result<Void> delete(@PathVariable Long id) {
        userService.deleteUser(id);
        return Result.success();
    }
}
```

## ✅ 优势总结

1. **统一返回格式** - 所有API返回格式一致
2. **统一异常处理** - 不需要try-catch，自动捕获并返回标准格式
3. **自动对象转换** - PO、DTO、VO之间转换简单快捷
4. **自动ID生成** - 使用雪花算法生成全局唯一ID
5. **自动字段填充** - 创建时间、更新时间自动填充
6. **代码简化** - 大幅减少样板代码
