# 注册接口使用说明

## 📌 重要说明

### ✅ 字段验证规则

**所有字段都是可选的！** 系统会根据 `registerType` 自动判断需要验证哪些字段：

| 注册方式 | 必填字段 | 可选字段 |
|---------|---------|---------|
| **phone** | `userType`, `registerType`, `phone`, `password`, `phoneVerifyCode` | `username`, `realName`, `inviteCode` 等 |
| **email** | `userType`, `registerType`, `email`, `password`, `emailVerifyCode` | `username`, `realName`, `inviteCode` 等 |
| **third_party** | `userType`, `registerType`, `thirdPartyPlatform`, `thirdPartyOpenId` | `thirdPartyUnionId`, `thirdPartyNickname`, `realName` 等 |

### ⚠️ 不要混用不同注册方式的字段！

例如，选择手机号注册时，**不要填写**邮箱字段或第三方字段。

---

## 🔧 正确的使用方式

### 方式1：手机号注册

```json
{
  "userType": "normal",
  "registerType": "phone",
  "username": "johndoe",
  "password": "pass123456",
  "confirmPassword": "pass123456",
  "phone": "13900139001",
  "phoneVerifyCode": "123456",
  "realName": "张三",
  "clientIp": "127.0.0.1"
}
```

**只需要这些字段：**
- ✅ `userType` - 用户类型（必填）
- ✅ `registerType` - 注册方式（必填，填 "phone"）
- ✅ `phone` - 手机号（必填）
- ✅ `password` - 密码（必填）
- ✅ `confirmPassword` - 确认密码（必填）
- ✅ `phoneVerifyCode` - 手机验证码（必填）
- ⚪ `username` - 用户名（可选）
- ⚪ `realName` - 真实姓名（可选）
- ⚪ `clientIp` - 客户端IP（可选）

**不要填写：**
- ❌ `email` - 邮箱字段（手机号注册不需要）
- ❌ `emailVerifyCode` - 邮箱验证码（手机号注册不需要）
- ❌ `thirdPartyPlatform` - 第三方平台（手机号注册不需要）
- ❌ `thirdPartyOpenId` - 第三方OpenID（手机号注册不需要）

- ✅ `confirmPassword` - 确认密码（必填）

- ⚪ `realName` - 真实姓名（可选）

### 方式3：第三方注册（微信）

```json
{
  "userType": "normal",
  "registerType": "third_party",
  "thirdPartyPlatform": "wechat",
  "thirdPartyOpenId": "oXXXXXXXXXXXXXXXX",
  "thirdPartyUnionId": "uXXXXXXXXXXXXXXXX",
  "thirdPartyNickname": "张三",
  "thirdPartyAvatar": "https://example.com/avatar.jpg",
  "realName": "张三",
  "clientIp": "127.0.0.1"
}
```

**只需要这些字段：**
- ✅ `userType` - 用户类型（必填）
- ✅ `registerType` - 注册方式（必填，填 "third_party"）
- ✅ `thirdPartyPlatform` - 第三方平台（必填，如 "wechat"）
- ✅ `thirdPartyOpenId` - 第三方OpenID（必填）
- ⚪ `thirdPartyUnionId` - 第三方UnionID（可选）
- ⚪ `thirdPartyNickname` - 第三方昵称（可选）
- ⚪ `thirdPartyAvatar` - 第三方头像（可选）
- ⚪ `realName` - 真实姓名（可选）

**注意：第三方注册不需要密码！**

## 🎯 4种用户身份

所有注册方式都支持4种用户身份：

1. **normal** - 普通用户
2. **merchant** - 商家
3. **operator** - 运营人员
4. **admin** - 管理员

只需修改 `userType` 字段即可。

## 🚀 快捷测试接口

Swagger 中提供了快捷测试接口：

### 手机号注册
- `POST /api/auth/test/register/phone/normal` - 普通用户手机号注册
- `POST /api/auth/test/register/phone/merchant` - 商家手机号注册

### 邮箱注册
- `POST /api/auth/test/register/email/normal` - 普通用户邮箱注册

### 手机号登录
- `POST /api/auth/test/login/phone/normal` - 普通用户手机号登录
- `POST /api/auth/test/login/phone/merchant` - 商家手机号登录

### 邮箱登录
- `POST /api/auth/test/login/email/normal` - 普通用户邮箱登录

**直接在 Swagger UI 中点击 "Try it out" 即可测试！**

## 💡 前端集成建议

### 前端实现步骤：

1. **第一步：选择用户身份**
   ```
   下拉框：[普通用户] [商家] [运营人员] [管理员]
   ```

2. **第二步：选择注册方式**
   ```
   Tab切换：[手机号注册] [邮箱注册] [微信注册]
   ```

3. **第三步：显示对应表单**

   **手机号注册表单：**
   - 手机号输入框
   - 验证码输入框
   - 密码输入框
   - 确认密码输入框

   **邮箱注册表单：**
   - 邮箱输入框
   - 验证码输入框
   - 密码输入框
   - 确认密码输入框

   **微信注册表单：**
   - 点击"微信授权登录"按钮
   - 自动获取微信信息
   - 无需手动输入

4. **第四步：提交对应的字段**

   根据选择的注册方式，只提交需要的字段，不要提交其他方式的字段。

## ⚠️ 常见错误

### 错误1：混用不同方式的字段

❌ **错误示例：**
```json
{
  "registerType": "phone",
  "phone": "13900139001",
  "email": "user@example.com",  // ❌ 手机号注册不需要邮箱！
  "password": "pass123456"
}
```

✅ **正确做法：**
```json
{
  "registerType": "phone",
  "phone": "13900139001",
  "password": "pass123456"
}
```

### 错误2：缺少必填字段

❌ **错误示例：**
```json
{
  "registerType": "phone",
  "phone": "13900139001"
  // ❌ 缺少 password 和 phoneVerifyCode
}
```

✅ **正确做法：**
```json
{
  "registerType": "phone",
  "phone": "13900139001",
  "password": "pass123456",
  "phoneVerifyCode": "123456"
}
```
