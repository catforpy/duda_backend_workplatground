# 认证接口使用说明

## 📌 三种注册/登录方式

### 方式1️⃣：都达网账号注册/登录

**特点**：传统的用户名+密码方式

**注册**：
```json
{
  "userType": "normal",
  "registerType": "account",
  "username": "johndoe",
  "password": "pass123456",
  "confirmPassword": "pass123456",
  "realName": "张三"
}
```

**登录**：
```json
{
  "loginType": "account_password",
  "userType": "normal",
  "username": "johndoe",
  "password": "pass123456"
}
```

---

### 方式2️⃣：手机号验证码注册/登录

**特点**：手机号+验证码，**不需要密码**

**注册**：
```json
{
  "userType": "normal",
  "registerType": "phone_sms",
  "phone": "13900139001",
  "phoneVerifyCode": "123456",
  "realName": "张三"
}
```

**登录**：
```json
{
  "loginType": "phone_sms",
  "userType": "normal",
  "phone": "13900139001",
  "phoneVerifyCode": "123456"
}
```

**注意**：这种方式不需要密码！类似滴滴、美团、京东的验证码登录。

---

### 方式3️⃣：微信扫码注册/登录

**特点**：扫码一次性完成注册和登录

**注册/登录**（一次性）：
```json
{
  "userType": "normal",
  "registerType": "wechat_scan",
  "thirdPartyPlatform": "wechat",
  "thirdPartyOpenId": "oXXXXXXXXXXXXXXXX",
  "thirdPartyUnionId": "uXXXXXXXXXXXXXXXX",
  "thirdPartyNickname": "张三",
  "thirdPartyAvatar": "https://example.com/avatar.jpg"
}
```

**登录**：
```json
{
  "loginType": "wechat_scan",
  "userType": "normal",
  "thirdPartyPlatform": "wechat",
  "thirdPartyOpenId": "oXXXXXXXXXXXXXXXX"
}
```

**注意**：微信方式会自动注册，不需要密码。

---

## 🎯 支持的4种用户身份

所有注册/登录方式都支持：

1. **normal** - 普通用户
2. **merchant** - 商家
3. **operator** - 运营人员
4. **admin** - 管理员

只需要修改 `userType` 字段即可。

---

## 📋 完整示例

### 示例1：普通用户 - 都达网账号注册

**请求**：
```bash
curl -X POST http://localhost:8083/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "userType": "normal",
    "registerType": "account",
    "username": "alice",
    "password": "pass123456",
    "confirmPassword": "pass123456",
    "realName": "艾丽斯"
  }'
```

**响应**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "userId": 1000123456789,
    "username": "alice",
    "userType": "normal",
    "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "tokenType": "Bearer",
    "expiresIn": 900
  }
}
```

### 示例3：管理员 - 微信扫码登录

**前端流程**：
1. 用户点击"微信登录"
2. 跳转到微信授权页面
3. 用户扫码授权
4. 微信回调获取 `openId` 和 `unionId`
5. 前端调用登录接口

**请求**：
```bash
curl -X POST http://localhost:8083/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "loginType": "wechat_scan",
    "userType": "admin",
    "thirdPartyPlatform": "wechat",
    "thirdPartyOpenId": "oWECHAT_OPEN_ID_123",
    "thirdPartyUnionId": "uWECHAT_UNION_ID_456",
    "thirdPartyNickname": "张三"
  }'
```

**响应**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userId": 1000123456789,
    "userType": "admin"
  }
}
```

## ⚠️ 重要说明

### 1. 字段验证规则

所有字段都是**可选的**，系统会根据 `registerType`/`loginType` 自动判断：

| 方式 | 必填字段 | 不需要 |
|------|---------|--------|
| **都达网账号** | username, password | phone, email, 第三方字段 |
| **手机验证码** | phone, phoneVerifyCode | password, username |
| **微信扫码** | thirdPartyPlatform, thirdPartyOpenId | password, phone |

### 2. 手机号验证码方式特点

- ✅ **不需要密码**，更方便
- ✅ 适合移动端
- ✅ 类似滴滴、美团、京东的登录方式
- ⚠️ 需要短信服务支持

### 3. 微信扫码方式特点

- ✅ 一次性完成注册和登录
- ✅ 无需记忆密码
- ✅ 适合微信生态
- ⚠️ 需要微信开放平台对接

## 🔧 Token 使用

### 获取Token后，在请求头中携带：

```bash
curl -X POST http://localhost:8083/api/auth/user/info \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### Token 类型：

- **Access Token**：短期令牌（15分钟），用于API调用
- **Refresh Token**：长期令牌（7天），用于刷新 Access Token

## 📝 总结

1. **三种方式，各有所长**
   - 都达网账号：传统方式，适合PC端
   - 手机验证码：便捷方式，适合移动端
   - 微信扫码：快捷方式，适合微信生态

2. **根据 registerType/loginType 选择对应字段**
   - 不要混用不同方式的字段
   - 参考上面的示例代码

3. **所有方式都支持4种用户身份**
   - normal、merchant、operator、admin

4. **使用快捷测试接口快速验证**
   - 在 Swagger UI 中直接测试
